How to Install Divi

  1. Go to Dashboard-> Divi -> Divi Library
  2. Click Import and Export button
  3. Click Import tab and click choose File button
  4. Select layout json file from downloaded Divi layout packages.Click import "divi builder layouts" button
  5. Once import successfully completed.Layout was displayed here.

How to Create Page & Import

  1. Go to Dashboard-> Pages -> Add New
  2. Enter Page title and then click use divi builder button
  3. Click Start Building button
  4. Click Start Building button
  5. Click Three dot circle icon
  6. Click Load from Library plus icon
  7. Click Your saved layouts tab.Click Your Layouts.After loaded layout -> click save button
